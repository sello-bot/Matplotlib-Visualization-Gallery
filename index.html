User Growth Chart

Track growth patterns over time with filled area under the line. Perfect for showing user acquisition or revenue growth trends.

import matplotlib.pyplot as plt

months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun']
users = [50, 80, 120, 180, 250, 320]
plt.fill_between(months, users, color="skyblue", alpha=0.4)
plt.plot(months, users, color="blue", marker="o")
plt.title("User Growth Over Time")
plt.xlabel("Months")
plt.ylabel("Number of Users")
plt.grid(True)
plt.show()

Performance Correlation

Visualize relationships between practice time and performance outcomes. Ideal for educational data analysis.

import matplotlib.pyplot as plt

x = [5, 7, 8, 7, 6, 9, 5, 8, 7, 6, 5]
y = [99, 86, 87, 100, 86, 103, 87, 103, 87, 94, 78]
plt.scatter(x, y, color='#ff5733')
plt.title("Hours of Coding vs Performance")
plt.xlabel("Hours of coding")
plt.ylabel("Performance Score")
plt.grid(True)
plt.show()

Sales Comparison

Compare performance across different time periods. Excellent for year-over-year analysis and growth tracking.

import matplotlib.pyplot as plt

months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun']
sales_2024 = [120, 150, 170, 160, 180, 210]
sales_2025 = [140, 160, 180, 200, 220, 250]
plt.plot(months, sales_2024, marker='o', label='2024 Sales')
plt.plot(months, sales_2025, marker='o', label='2025 Sales')
plt.title("Sales Growth Comparison")
plt.xlabel("Month")
plt.ylabel("Sales (in $1000)")
plt.legend()
plt.grid(True)
plt.show()

Bar Chart

Compare categorical data with rectangular bars. Perfect for showing sales by category, survey results, or any discrete comparisons.

import matplotlib.pyplot as plt

categories = ['Electronics', 'Clothing', 'Books', 'Food']
sales = [12000, 8500, 4500, 9800]

plt.bar(categories, sales, color=['#8b5cf6', '#a78bfa', '#c4b5fd', '#ddd6fe'])
plt.title('Sales by Category')
plt.xlabel('Product Category')
plt.ylabel('Sales ($)')
plt.xticks(rotation=45)
plt.grid(axis='y', alpha=0.3)
plt.show()

Pie Chart

Show proportions and percentages of a whole. Great for market share, budget allocation, or demographic breakdowns.

import matplotlib.pyplot as plt

labels = ['Apple', 'Samsung', 'Google', 'Other']
market_share = [45, 25, 15, 15]
colors = ['#ff6b6b', '#4ecdc4', '#45b7d1', '#f9ca24']

plt.pie(market_share, labels=labels, colors=colors, autopct='%1.1f%%')
plt.title('Smartphone Market Share')
plt.axis('equal')
plt.show()

Histogram

Display the distribution of a continuous variable. Useful for understanding data spread, central tendency, and outliers.

import matplotlib.pyplot as plt
import numpy as np

ages = np.random.normal(35, 12, 1000)
plt.hist(ages, bins=20, color='teal', alpha=0.7, edgecolor='black')
plt.title('Age Distribution')
plt.xlabel('Age')
plt.ylabel('Frequency')
plt.grid(alpha=0.3)
plt.show()

Stacked Area Chart

import matplotlib.pyplot as plt
import numpy as np

months = ['Jan', 'Feb', 'Mar', 'Apr', 'May']
product_a = [20, 35, 30, 35, 27]
product_b = [25, 32, 34, 20, 25]
product_c = [15, 18, 29, 31, 17]

plt.stackplot(months, product_a, product_b, product_c,
labels=['Product A', 'Product B', 'Product C'],
alpha=0.8)
plt.legend(loc='upper left')
plt.title('Monthly Sales by Product')
plt.xlabel('Month')
plt.ylabel('Sales')
plt.show()

Tips for Better Visualizations

  • Always label your axes and include a title
  • Use appropriate colors and color palettes
  • Consider your audience and choose the right plot type
  • Use grid lines sparingly to improve readability
  • Add legends when showing multiple data series
  • Adjust figure size for better presentation: plt.figure(figsize=(10, 6))
  • Save high-quality images: plt.savefig('plot.png', dpi=300, bbox_inches='tight')
